# web-FS22 # 🌾 Sistema de Postulaciones - FS22 Vanilla RP

Esta es una aplicación web sencilla diseñada para que los aspirantes a **Staff** del servidor de Farming Simulator 22 (Vanilla RP) puedan enviar sus solicitudes directamente a nuestro canal de Discord.

## 🚀 Características
- **Interfaz limpia:** Diseño oscuro inspirado en la estética de Discord.
- **Formulario interactivo:** Recoge el nombre de usuario, edad y motivación del candidato.
- **Integración con Discord:** Envío en tiempo real mediante un Webhook.

## 🛠️ Tecnologías utilizadas
- **HTML5:** Estructura de la página.
- **CSS3:** Diseño y estilos personalizados.
- **JavaScript:** Lógica de envío y conexión con la API de Discord.

## 📂 Estructura del proyecto
- `index.html`: El archivo principal de la web.
- `style.css`: La hoja de estilos para los colores y el diseño.
- `script.js`: El cerebro que envía los datos a Discord.

## ⚙️ Instalación y Uso
1. Clona este repositorio o descarga los archivos.
2. Abre el archivo `script.js` y coloca tu **Webhook URL** de Discord en la variable `webhookURL`.
3. ¡Listo! Abre el `index.html` en tu navegador o súbelo a GitHub Pages.
